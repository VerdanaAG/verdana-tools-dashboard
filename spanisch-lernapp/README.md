# Verdana Spanisch Coach

MVP einer webbasierten Spanisch-Lernapp fuer absolute Anfaenger.

## Ziel

Die App vermittelt spanische Grundlagen und priorisiert Wortschatz fuer Verdana AG relevante Situationen:

- Begruessung und Alltag
- Kundengespraeche
- Meetings
- Produkte und Materialien
- Qualitaet, Lieferung und Projekte
- einfache Business-Kommunikation

## Start

`index.html` im Browser oeffnen.

## Struktur

- `index.html` UI-Struktur
- `styles.css` Design
- `app.js` Lektionen, Quiz, Flashcards und Fortschritt
- `manifest.webmanifest` Vorbereitung als installierbare Web-App
